## Project-Specific Planner Guidance

For game development projects like the Snake game, the Planner should consider the following phases:
- Concept Development: Define game mechanics, features, and user experience.
- Design: Create game design documents, wireframes, and asset requirements.
- Implementation: Plan the development of core components such as game loop, physics, graphics, and controls.
- Testing: Schedule playtesting sessions to gather feedback and identify bugs.
- Deployment: Prepare for distribution across various platforms and ensure compatibility.

Focus on iterative development with regular milestones to allow for continuous testing and refinement.
